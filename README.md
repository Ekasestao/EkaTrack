# EkaWatch

Proyecto personal de seguimiento de películas y series, desarrollado por hobby sin ningún fin comercial ni lucrativo. Inspirado en la plataforma [JustWatch](https://www.justwatch.com).

Permite buscar títulos, organizarlos en listas, marcar lo que has visto y descubrir nuevos estrenos.

> Este proyecto utiliza la API de TMDB pero no está respaldado ni certificado por TMDB.

## Estado

En desarrollo activo. Funcionalidades principales implementadas y funcionando.

## Aviso

Este es un proyecto personal con fines exclusivamente educativos y de entretenimiento propio. Inspirado en JustWatch, sin ninguna relación oficial con dicha plataforma. No se comercializa, no se monetiza y no se recogen datos con ningún fin lucrativo.
